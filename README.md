# hello-psram

#PSRAM入門

## 概要

ESP32シリーズで使われる PSRAM (Pseudo Static Random Access Memory)。
我々捜査班はPSRAMの謎に迫るべくリポジトリを作成した...。

## ビルド方法

~~~
idf.py build
~~~

## 実行方法

~~~
idf.py flash monitor
~~~
